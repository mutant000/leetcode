## Notes

- `231 Power of Two`: 正数如果是 2 的幂，二进制里只有一个 1，用 `n & (n - 1)` 清掉最低位 1 后应为 0。
- `232 Implement Queue using Stacks`: 一个栈负责入队，另一个栈负责出队，需要读取队首时再把元素倒过去。
- `233 Number of Digit One`: 按个位、十位、百位逐层统计当前位为 1 的贡献。
- `235 Lowest Common Ancestor of a Binary Search Tree`: 利用 BST 大小关系，两个节点都在一侧就继续走，否则当前节点就是最近公共祖先。
- `237 Delete Node in a Linked List`: 不能访问前驱时，把下一个节点的值复制过来，再跳过下一个节点。
- `241 Different Ways to Add Parentheses`: 遇到运算符就把表达式分成左右两边，递归合并所有可能结果，并用缓存避免重复计算。
- `258 Add Digits`: 数字根规律，`0` 返回 `0`，其他数用 `(num - 1) % 9 + 1`。
- `260 Single Number III`: 先异或得到两个特殊数的差异位，再按最低差异位分组异或。
- `263 Ugly Number`: 只要能被 2、3、5 整除就持续除，最后剩 1 才是 ugly number。
- `264 Ugly Number II`: 三个指针分别生成乘 2、3、5 的候选数，每次取最小并推进对应指针。
