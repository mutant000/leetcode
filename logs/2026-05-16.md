## 131. Palindrome Partitioning

- Track: LeetCode Top 100 Liked
- Difficulty: Medium
- Topic: String, Dynamic Programming, Backtracking
- Status: Locally implemented and tested; LeetCode submission still pending
- Submission: Not available

## 思路

先用 `is_palindrome[i][j]` 预处理每个子串是否为回文，再用 DFS 枚举所有合法切分。

回溯到位置 `start` 时，尝试所有结尾 `end`：

- 如果 `s[start:end+1]` 是回文，就加入当前路径。
- 递归处理后缀。
- 返回时撤销这段子串。

这样可以避免在回溯过程中反复线性判断同一段子串是不是回文。

## 26. Remove Duplicates from Sorted Array

- Track: LeetCode Top 100 Liked
- Difficulty: Easy
- Topic: Array, Two Pointers
- Status: Accepted, 362 / 362 testcases passed
- Submission: https://leetcode.com/problems/remove-duplicates-from-sorted-array/submissions/2004257203/
- Runtime: 0 ms, Beats 100.00%
- Memory: 20.71 MB, Beats 16.66%

## 思路

因为数组已经有序，重复值一定相邻。

用 `write` 维护去重结果的末尾位置，从左到右扫描每个数：

- 如果当前值和 `nums[write - 1]` 不同，说明是一个新值。
- 把它写到 `nums[write]`，再把 `write` 向后移动。

最后前 `write` 个元素就是去重后的答案，返回 `write`。
