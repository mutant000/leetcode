## Notes

- `424 Longest Repeating Character Replacement`: 用滑动窗口维护最高字符频率，窗口长度减去最高频率不超过替换次数时即可保留。
- `427 Construct Quad Tree`: 用二维前缀和常数时间判断区域是否全相同，再递归构造四个象限。
- `429 N-ary Tree Level Order Traversal`: 使用队列按当前层节点数分层处理 N 叉树。
- `430 Flatten a Multilevel Doubly Linked List`: 用栈模拟 DFS，先保存 `next` 再处理 `child`，并重新连接双向指针。
- `432 All O-one Data Structure`: 使用哈希表定位键所在桶，用有序双向桶链表常数时间维护最小和最大计数。
- `434 Number of Segments in a String`: 统计所有非空格片段的起始字符。
- `435 Non-overlapping Intervals`: 按终点排序并优先保留结束最早的区间。
- `436 Find Right Interval`: 排序所有起点，对每个终点二分查找第一个不小于它的起点。
- `440 K-th Smallest in Lexicographical Order`: 把数字视作前缀树，通过统计子树大小整段跳过字典序区间。
- `441 Arranging Coins`: 二分查找满足三角数不超过硬币总数的最大行数。
- `442 Find All Duplicates in an Array`: 把值映射到数组下标并用正负号标记，第二次访问负数位置时记录重复值。
- `443 String Compression`: 使用读写指针按连续字符分组，并把多位次数原地写回数组。
- `445 Add Two Numbers II`: 用两个栈从最低位开始相加，再把新节点依次插入结果链表头部。
- `446 Arithmetic Slices II - Subsequence`: 动态记录每个位置、每种公差下可继续扩展的子序列数量。
- `447 Number of Boomerangs`: 对每个中心点统计距离平方，相同距离的点产生 `count * (count - 1)` 个有序组合。
- `448 Find All Numbers Disappeared in an Array`: 用对应下标的符号记录出现过的数字，最后收集仍为正数的位置。
- `449 Serialize and Deserialize BST`: 迭代记录前序序列，再用单调祖先栈恢复 BST。
- `450 Delete Node in a BST`: 迭代定位节点；有两个孩子时用中序后继替换并删除后继。
- `451 Sort Characters By Frequency`: 统计字符频率并按出现次数从高到低拼接。
- `453 Minimum Moves to Equal Array Elements`: 将增加 `n - 1` 个元素等价转换为减少一个元素，累加各元素与最小值的差。
