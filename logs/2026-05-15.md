## 217. Contains Duplicate

- Track: LeetCode Top 100 Liked
- Difficulty: Easy
- Topic: Hash Table
- Status: Accepted, submitted on LeetCode
- Submission: https://leetcode.com/submissions/detail/2003530631/
- Note: Resubmitted in Python3 and accepted.

## 思路

用哈希集合记录已见数字。

按元素遍历数组，若当前值已在集合中则发现重复并返回 `True`；否则加入集合。
遍历结束说明全部不重复，返回 `False`。

## 242. Valid Anagram

- Track: LeetCode Top 100 Liked
- Difficulty: Easy
- Topic: Hash Table
- Status: Accepted, submitted on LeetCode
- Submission: https://leetcode.com/submissions/detail/2003512582/
- Note: Accepted submission verified from recent AC list in the logged-in LeetCode profile.

## 思路

先统计 `s` 中每个字符出现次数，再遍历 `t` 做抵消。

若 `t` 中出现了 `s` 没有的字符，或某字符出现次数超出 `s`，可提前返回 `False`。
最终所有计数都回到 0，表示两字符串字符全集与频率完全一致，返回 `True`。

## 88. Merge Sorted Array

- Track: LeetCode Top 100 Liked
- Difficulty: Easy
- Topic: Array, Two Pointers
- Status: Accepted, submitted on LeetCode
- Submission: https://leetcode.com/submissions/detail/2003531672/
- Note: Submitted in Python3 and accepted.

## 思路

从后往前用 `nums1`、`nums2` 的最后有效下标比较，较大值写入 `nums1` 的尾部。

当某一边取尽后，只需把 `nums2` 的剩余前缀补齐到前面；`nums1` 的剩余部分天然有序可保留。


## 102. Binary Tree Level Order Traversal

- Track: LeetCode Top 100 Liked
- Difficulty: Medium
- Topic: Tree, BFS, Queue
- Status: Accepted, submitted on LeetCode
- Submission: https://leetcode.com/submissions/detail/2003534562/
- Note: Re-submitted with clipboard paste in Python3 and accepted.

## 思路

用广度优先遍历按层推进。

用 `deque` 作为队列，记录每一层开始时队列长度 `level_size`。

一次只处理该层节点数量，把它们的值写入 `level`，并把下一层节点加入队列。全部处理完后把 `level` 追加到答案。

## 19. Remove Nth Node From End of List

- Track: LeetCode Top 100 Liked
- Difficulty: Medium
- Topic: Linked List, Two Pointers
- Status: Accepted, submitted on LeetCode
- Submission: https://leetcode.com/submissions/detail/2003534890/
- Note: Re-submitted with clipboard paste in Python3 and accepted.

## 思路

使用带 `dummy` 的双指针让 `right` 先领先 `left` `n + 1` 步。

同步移动时，当 `right` 到达 `None` 时，`left.next` 就是要删除的倒数第 `n` 个节点。执行
`left.next = left.next.next` 即可删除。
