# 300. Longest Increasing Subsequence

## 思路

维护数组 `tails`，其中 `tails[length - 1]` 表示长度为 `length` 的递增子序列能够拥有的最小结尾值。

遍历每个数字，用二分找到它应该替换的位置：

- 如果它大于所有结尾值，就能扩展出更长的递增子序列。
- 否则替换第一个大于等于它的结尾值，让同长度子序列的结尾尽量小。

`tails` 的长度就是最长递增子序列长度。

## 复杂度

- Time: `O(n log n)`
- Space: `O(n)`

## 边界

- 严格递增数组会不断追加。
- 严格递减数组会不断替换第一个位置。
- 重复数字不能延长严格递增序列，因此使用 `bisect_left`。
