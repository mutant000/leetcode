# 763. Partition Labels

## 思路

先记录每个字符最后一次出现的位置。

从左到右扫描当前片段，片段右边界 `end` 必须覆盖片段内所有字符的最后位置。每看到一个字符，就用它的最后位置更新 `end`。当扫描下标等于 `end` 时，当前片段已经封闭，可以记录长度并从下一个位置开始新片段。

## 复杂度

- Time: `O(n)`。
- Space: `O(1)`，字符集固定为小写英文字母；按实现看哈希表大小与不同字符数有关。

## 边界

- 所有字符互不重复时，每个字符单独成段。
- 一个字符跨越很远时，片段会被延长到它最后一次出现的位置。
- 最后一段在扫描到字符串末尾时自然闭合。
