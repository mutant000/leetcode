# 160. Intersection of Two Linked Lists

## 思路

用两个指针分别从 `headA` 和 `headB` 出发。

1. 每个指针每次走一步。
2. 指针走到自己链表末尾后，切换到另一条链表的头。
3. 如果两条链表相交，两个指针走过的总长度都会是 `lenA + lenB`，会在交点相遇。
4. 如果不相交，两个指针最终都会走到 `None`，循环结束并返回 `None`。

这个方法不需要提前计算链表长度，也不会修改原链表结构。

## 复杂度

- Time: `O(m + n)`
- Space: `O(1)`

## 边界

- 任意一个链表为空，直接最终返回 `None`。
- 两条链表不相交时，两个指针会同步到达 `None`。
- 交点可能就是某条链表的头节点。
