# 14. Longest Common Prefix

## 思路

先把第一个字符串当作当前公共前缀 `prefix`。

之后依次拿它和后面的字符串比较：

- 如果当前字符串不是以 `prefix` 开头，就不断把 `prefix` 去掉最后一个字符。
- 缩短到能匹配当前字符串为止。
- 如果 `prefix` 被缩成空串，说明所有字符串没有公共前缀，直接返回 `""`。

这样做的本质是不断维护“到目前为止所有字符串的最长公共前缀”。

## 复杂度

- Time: `O(S)`，其中 `S` 是所有字符串字符总数；每个字符最多被比较和裁剪有限次。
- Space: `O(1)`，只使用了若干额外变量。

## 边界

- 只有一个字符串时，答案就是它本身。
- 任意字符串为空串时，公共前缀最多只能是空串。
- 第一位字符就不同，例如 `["dog", "racecar", "car"]`，会直接返回 `""`。
- 某个字符串本身就是最短公共前缀，例如 `["flower", "flow", "flight"]`，答案是 `"fl"`。
