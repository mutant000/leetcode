# 26. Remove Duplicates from Sorted Array

## 思路

数组已经是升序，所以重复元素一定连续出现。

用双指针原地去重：
- `read` 负责从左到右扫描每个元素
- `write` 指向下一个应该写入新值的位置

先保留第一个元素，从第二个元素开始遍历。

如果 `nums[read]` 和当前去重后末尾 `nums[write - 1]` 不同，说明遇到了一个新的数字，就把它写到 `nums[write]`，再把 `write` 往后移一格。

遍历结束后，前 `write` 个位置就是去重后的结果，返回 `write` 即可。

## 复杂度

- Time: `O(n)`
- Space: `O(1)`

## 边界

- 空数组时直接返回 `0`。
- 只有一个元素时直接返回 `1`。
- 全部元素都相同时，结果长度为 `1`。
- 没有重复元素时，数组保持原顺序不变。
