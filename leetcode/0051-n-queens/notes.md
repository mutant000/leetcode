# 51. N-Queens

## 思路

按行回溯放皇后，每一行只尝试一个位置。

用三个集合分别记录已经被占用的列、主对角线 `row + column`、副对角线 `row - column`。这样判断当前位置是否合法只需要 `O(1)`。

当 `row == n` 时，说明前 `n` 行都已经成功放置皇后，此时根据每一行保存的列位置还原棋盘并加入答案。

## 复杂度

- Time: `O(n!)`，回溯会枚举满足约束的列摆放方案，最坏情况下数量接近全排列级别。
- Space: `O(n)`，递归深度、列记录和对角线记录都与 `n` 成正比；结果列表不计入额外空间。

## 边界

- `n = 1` 时唯一解是 `["Q"]`。
- `n = 2` 和 `n = 3` 时无解，回溯结束后返回空列表。
- 返回顺序不限，只要每个棋盘都满足互不攻击即可。
