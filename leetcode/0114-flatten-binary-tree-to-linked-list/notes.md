# 114. Flatten Binary Tree to Linked List

## 思路

目标链表顺序是前序遍历：根、左、右。

可以反过来处理：右、左、根。用 `previous` 保存已经整理好的链表头：
- 先递归整理右子树。
- 再递归整理左子树。
- 当前节点的 `right` 指向 `previous`，`left` 置空。
- 然后把当前节点变成新的 `previous`。

这样回到根节点时，整棵树已经被原地拉平成只走右指针的链表。

## 复杂度

- Time: `O(n)`
- Space: `O(h)`

## 边界

- 空树。
- 只有一个节点。
- 只有左子树或只有右子树。
