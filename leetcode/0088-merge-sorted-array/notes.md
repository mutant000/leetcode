# 88. Merge Sorted Array

## 思路

题目要求在 `nums1` 的前 `m+n` 长度内得到两个升序数组的并集结果，并且不能使用额外数组。

可以用三个指针从后往前覆盖：
- `i = m - 1` 指向 `nums1` 的有效区末尾
- `j = n - 1` 指向 `nums2` 的末尾
- `k = m + n - 1` 指向 `nums1` 的最终填充位置

每次比较 `nums1[i]` 和 `nums2[j]`，把更大的数写到 `nums1[k]`，然后向前移动对应指针和 `k`。

当 `nums2` 还有剩余元素时，直接继续填到 `nums1` 的前段。`nums1` 剩下的元素若还在前面，天然已经有序，不需要移动。

## 复杂度

- Time: `O(m + n)`
- Space: `O(1)`

## 边界

- `nums2` 为空时，`nums1` 不变。
- `nums1` 中有效元素为空（`m == 0`）时，直接把 `nums2` 复制到 `nums1`。
- 有重复值时，比较和写入时保持稳定性；算法天然满足。
