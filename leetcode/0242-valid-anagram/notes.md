# 242. Valid Anagram

## 思路

两个字符串长度不同则一定不是异位词。

用字典统计 `s` 中每个字符出现次数，再遍历 `t` 做抵消：

1. 每出现一个字符就加 1。
2. 遍历 `t` 时，对应字符减 1；出现负数或不存在说明多出字符，直接返回 `False`。
3. 所有字符匹配后，字典中剩余计数应清空，返回 `True`。

## 复杂度

- Time: `O(n)`，`n` 为字符串长度
- Space: `O(k)`，`k` 为字符集合大小（最坏 `O(n)`）

## 边界

- 空字符串和空字符串返回 `True`。
- 长度不同直接 `False`。
- 字符重复、全部相同字符或完全不同字符都能正确处理。
