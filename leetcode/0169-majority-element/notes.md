# 169. Majority Element

## 思路

使用 Boyer-Moore 投票法。

维护一个候选数 `candidate` 和计数 `count`。从左到右扫描数组：

1. 如果 `count == 0`，说明之前的候选数已经和其他数字两两抵消，当前数字成为新的候选数。
2. 当前数字等于候选数时，`count += 1`。
3. 当前数字不等于候选数时，`count -= 1`。

因为题目保证多数元素出现次数超过 `n / 2`，它和其他所有元素抵消后仍会留下来，最终候选数就是答案。

## 复杂度

- Time: `O(n)`
- Space: `O(1)`

## 边界

- 数组只有一个元素时，循环后直接返回该元素。
- 多数元素可能在开头、中间或末尾，投票法不依赖位置。
- 数字可能是负数或很大，只做相等比较，不受数值范围影响。
