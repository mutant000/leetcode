# 198. House Robber

## 思路

每间房只有偷或不偷两种选择。如果偷当前房，就不能偷前一间；如果不偷当前房，就保留前一间为止的最优结果。

用两个变量滚动保存：

- `best_without_previous`：处理到前一间之前的最优值。
- `best_with_or_without_previous`：处理到前一间为止的最优值。

遍历到当前金额时，新的最优值就是 `max(不偷当前, 偷当前)`。

## 复杂度

- Time: `O(n)`
- Space: `O(1)`

## 边界

- 只有一间房。
- 相邻大值不能同时选择。
- 全部金额较小时仍按同一转移处理。
