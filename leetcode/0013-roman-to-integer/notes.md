# 13. Roman to Integer

## 思路

罗马数字通常从大到小排列，只有 `IV`、`IX`、`XL`、`XC`、`CD`、`CM` 这类小值在大值前面时需要做减法。

从左到右扫描字符串：

- 如果当前字符的值小于右边字符，说明它属于减法组合，把当前值减掉。
- 否则把当前值加到答案里。

这样不需要单独枚举六种特殊组合，因为“比右边小”正好覆盖所有合法减法情况。

## 复杂度

- Time: `O(n)`
- Space: `O(1)`

## 边界

- 单字符，例如 `I`、`V`、`M`。
- 连续加法，例如 `III`、`LVIII`。
- 多个减法组合，例如 `MCMXCIV`。
- 题目保证输入是 `[1, 3999]` 内的合法罗马数字。
