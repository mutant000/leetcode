# 136. Single Number

## 思路

利用异或的三个性质：

1. `x ^ x = 0`，相同数字会互相抵消。
2. `x ^ 0 = x`，零不会改变结果。
3. 异或满足交换律和结合律，数组顺序不影响最终结果。

把所有数字从左到右异或一遍，成对出现的数字都会被抵消，最后剩下的就是只出现一次的数字。

## 复杂度

- Time: `O(n)`
- Space: `O(1)`

## 边界

- 数组只有一个元素时，初始 `0` 与该元素异或后直接返回它。
- 唯一数字可能是负数，Python 的按位异或仍然满足抵消性质。
- 重复数字不要求相邻，异或的交换律保证顺序无关。
