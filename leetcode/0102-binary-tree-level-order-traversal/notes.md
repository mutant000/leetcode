# 102. Binary Tree Level Order Traversal

## 思路

层序遍历要求按层从上到下输出节点值。用队列（BFS）记录当前层的所有节点。

每轮取出队列长度作为当前层的节点数，只处理这些节点即可形成完整一层：
- 弹出节点并记录值到本层数组
- 将该节点的左、右孩子（若存在）追加到队列

处理完一层后把本层结果放入最终答案，再继续下一层。这样天然保证层序。

## 复杂度

- Time: `O(n)`
- Space: `O(n)`

## 边界

- 空树返回空列表 `[]`。
- 只有根节点时，返回 `[[root.val]]`。
- 单边链状树时，层数变长但每层只有 1 个节点，也能正确返回。
