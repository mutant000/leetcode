# 139. Word Break

## 思路

用动态规划判断前缀是否可以被字典拆分。

`can_break[end]` 表示 `s[:end]` 是否可拆分。枚举最后一个单词长度，如果前面的前缀可拆分，并且 `s[start:end]` 在字典中，就能推出当前前缀可拆分。

先把字典放入集合，并记录最大单词长度，枚举时只检查可能的后缀长度。

## 复杂度

- Time: `O(n * L)`，`L` 是字典中最大单词长度。
- Space: `O(n + w)`，`w` 是字典大小。

## 边界

- 整个字符串本身就是一个词。
- 需要重复使用字典中的词。
- 有些前缀可拆但最终整体不可拆，需要看完整 `can_break[-1]`。
