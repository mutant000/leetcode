# 236. Lowest Common Ancestor of a Binary Tree

## 思路

用 DFS 从根节点往下找 `p` 和 `q`。

递归函数返回当前子树里找到的目标节点：
- 如果当前节点为空，返回空。
- 如果当前节点就是 `p` 或 `q`，直接返回当前节点。
- 分别去左子树和右子树找。

如果左右子树都找到了目标，说明当前节点就是最近公共祖先。否则返回非空的那一边。

## 复杂度

- Time: `O(n)`
- Space: `O(h)`

## 边界

- `p` 是 `q` 的祖先。
- `q` 是 `p` 的祖先。
- 两个节点分别在根节点左右两侧。
