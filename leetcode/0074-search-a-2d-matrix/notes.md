# 74. Search a 2D Matrix

## 思路

题目保证每一行升序，并且下一行的第一个数大于上一行最后一个数，所以整个矩阵可以当作一个拉平后的一维升序数组。

对下标范围 `0` 到 `m * n - 1` 做二分：
- 一维下标 `mid` 对应二维位置 `row = mid // n`，`col = mid % n`。
- 如果当前位置小于目标，去右半边找。
- 如果当前位置大于目标，去左半边找。

## 复杂度

- Time: `O(log(mn))`
- Space: `O(1)`

## 边界

- 目标在第一个或最后一个位置。
- 目标不存在。
- 只有一行或一列。
