# 78. Subsets

## 思路

用回溯按顺序决定每个元素是否进入当前子集。

`path` 表示当前已经选中的元素，`start` 表示下一次只能从哪个下标开始继续选，避免生成重复子集。每进入一层递归，就先把当前 `path` 的拷贝加入答案；然后依次尝试把后续元素加入 `path`，递归处理，再撤销选择。

因为题目保证 `nums` 中元素互不相同，所以按下标递增展开后，每个子集只会生成一次。

## 复杂度

- Time: `O(n * 2^n)`，一共有 `2^n` 个子集，每次加入答案时复制当前路径需要 `O(n)`。
- Space: `O(n)`，递归深度和当前路径最多都是 `n`；结果列表不计入额外空间。

## 边界

- `nums.length = 1` 时，答案就是空集和只含该元素的集合。
- 数组里可能出现负数或 `0`，不会影响回溯逻辑。
- 题目保证元素互不相同，因此不需要额外去重。
