# 22. Generate Parentheses

## 思路

用回溯逐步构造合法括号串。

维护当前路径 `path`，以及已经放入的左括号数量 `open_count` 和右括号数量 `close_count`：

1. 只要左括号还没用满 `n` 个，就可以继续放 `"("`。
2. 只有当右括号数量小于左括号数量时，才可以放 `")"`，这样任何前缀都不会非法。
3. 当路径长度达到 `2 * n` 时，得到一个完整合法组合。

这个剪枝直接避免生成无效字符串。

## 复杂度

- Time: `O(C_n * n)`，`C_n` 是第 `n` 个卡特兰数，每个答案长度为 `2n`。
- Space: `O(C_n * n)`，结果列表保存所有合法组合；递归路径额外 `O(n)`。

## 边界

- `n = 1` 时只能生成 `["()"]`。
- 右括号不能先于左括号放入，否则前缀会非法。
- `n <= 8`，回溯枚举所有合法组合可以接受。
