# 283. Move Zeroes

## 思路

用 `write` 指针记录下一个非零元素应该写入的位置。

从左到右扫描原数组，遇到非零数就写到 `write`，然后把 `write` 右移。扫描结束后，`[0, write)` 已经按原相对顺序保存了所有非零数，剩下的位置全部补成 `0`。

整个过程只在原数组上修改，不创建额外数组。

## 复杂度

- Time: `O(n)`
- Space: `O(1)`

## 边界

- 全是 `0` 时，第一轮不会移动 `write`，第二轮把所有位置写成 `0`。
- 没有 `0` 时，第二轮不会执行，数组保持原顺序。
- 只有一个元素时，无论是 `0` 还是非零数都能正确保留。
- 负数也是非零元素，应该和正数一样保持相对顺序。
