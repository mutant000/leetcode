# 420. Strong Password Checker

## 思路

先统计缺少的小写字母、大写字母、数字类型数，以及每个连续重复段需要的替换次数 `length // 3`。

- 长度小于 6：插入操作既能补足长度，也能补字符类型或打断重复，答案是缺失类型数与缺少长度的较大值。
- 长度在 6 到 20：只需在补字符类型与打断重复之间取较大值。
- 长度大于 20：必须先删除。对长度模 3 为 0 的重复段删除 1 个字符、模 3 为 1 的重复段删除 2 个字符，都能优先减少一次替换；之后每删除 3 个字符可再减少一次替换。

## 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`
